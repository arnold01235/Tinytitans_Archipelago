# Mathias — Stardew Valley

## What you need

- **Stardew Valley** installed via Steam
- **SMAPI** (Stardew Valley mod loader): [smapi.io](https://smapi.io)
- The **Archipelago client** installed: [archipelago.gg/downloads](https://archipelago.gg/downloads)

## Step 1 — Install SMAPI

1. Download SMAPI from [smapi.io](https://smapi.io)
2. Run the installer and follow the instructions
3. After installation, always launch Stardew Valley through SMAPI (there will be a new shortcut)

## Step 2 — Install the Archipelago mod

1. Download the **Stardew Valley Archipelago mod** — it is included in the Archipelago client installation
   - After installing the Archipelago client, look in the client folder for a `Stardew Valley` folder
   - Alternatively, find it on the Archipelago releases page: [github.com/ArchipelagoMW/Archipelago/releases](https://github.com/ArchipelagoMW/Archipelago/releases)
2. Copy the mod folder into your Stardew Valley `Mods` folder:
   - Typically: `Steam\steamapps\common\Stardew Valley\Mods\`

## Step 3 — Configure your YAML

1. Open the Archipelago launcher
2. Go to **Generate Template Settings** and select Stardew Valley
3. Open the `.yaml` file in a text editor
4. Set `name` to your slot name (e.g. `Mathias`)
5. Set your preferred options (starting season, goal, friendship requirements, etc.)
6. Send the `.yaml` file to the host

## Step 4 — Connect to the server

1. Launch Stardew Valley through **SMAPI** (not directly from Steam)
2. Start a **new game** — Archipelago requires a fresh save, not an existing one
3. In-game, open the Archipelago menu (check the mod's keybind — default is usually `F5` or via the toolbar)
4. Enter the server address, slot name (`Mathias`), and password
5. Click **Connect**

## Tips

- You must start a new game; existing saves are not compatible
- Shipping items, completing bundles, and leveling skills are your checks
- You may receive seeds, tools, or resources from the multiworld early on
- Always launch through SMAPI or the mod won't load
