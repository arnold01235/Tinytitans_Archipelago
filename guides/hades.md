# Matnar — Hades

## What you need

- **Hades** installed via Steam or Epic Games
- **r2modman** or **Thunderstore Mod Manager**: [thunderstore.io](https://www.overwolf.com/app/Thunderstore-Thunderstore_Mod_Manager)
- The **Archipelago client** installed: [archipelago.gg/downloads](https://archipelago.gg/downloads)

## Step 1 — Install the mod manager

1. Download and install **r2modman**
2. Open it and select **Hades** as your game

## Step 2 — Install the Archipelago mod

1. In r2modman, go to the **Online** tab and search for `Archipelago`
2. Install the **Hades Archipelago** mod (dependencies will install automatically)
3. Make sure all mods are enabled

## Step 3 — Configure your YAML

1. Open the Archipelago launcher
2. Go to **Generate Template Settings** and select Hades
3. Open the `.yaml` file in a text editor
4. Set `name` to your slot name (e.g. `Matnar`)
5. Adjust options like goal (number of clears), boons, etc.
6. Send the `.yaml` file to the host

## Step 4 — Connect to the server

1. Launch Hades through r2modman (**Start modded**)
2. A connection window should appear — enter the server address, slot name (`Matnar`), and password
3. Click **Connect** and start your run

## Tips

- Each escape attempt is a run — checks are awarded by clearing rooms and bosses
- Boons and upgrades may come from the multiworld instead of the shop
- You don't have to escape on every run; progress still counts
- Launch through r2modman every time, not directly from Steam
