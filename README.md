# Tinytitans Archipelago Setup Guides

Welcome to our Archipelago multiworld! Below are setup guides for each player.

## What is Archipelago?

[Archipelago](https://archipelago.gg) is a multiworld randomizer platform where everyone plays their own game, but item pickups can be in each other's worlds. When you find an item, it might belong to someone else — and someone else finding an item might unlock something in your game.

## Common First Steps (Everyone)

1. Download and install the **Archipelago client** from [archipelago.gg/downloads](https://archipelago.gg/downloads)
2. Follow your game-specific guide (linked above)
3. Get your `.yaml` settings file ready — your guide explains what to set
4. Send your `.yaml` file to whoever is hosting the game session
5. When the session starts, launch your game client and connect to the server
