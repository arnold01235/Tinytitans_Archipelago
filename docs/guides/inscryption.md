# Adrian — Inscryption

## What you need

- **Inscryption** installed via Steam
- **r2modman** or **Thunderstore Mod Manager** — recommended mod manager: [thunderstore.io](https://www.overwolf.com/app/Thunderstore-Thunderstore_Mod_Manager)
- The **Archipelago client** installed: [github.com/ArchipelagoMW/Archipelago/releases](https://github.com/ArchipelagoMW/Archipelago/releases)

## Step 1 — Install the mod manager

1. Download and install **r2modman** from [Thunderstore](https://www.overwolf.com/app/Thunderstore-Thunderstore_Mod_Manager)
2. Open r2modman and select **Inscryption** as your game

## Step 2 — Install the Archipelago mod

1. In r2modman, go to **Online** tab and search for `Archipelago`
2. Install the **Archipelago** mod (it will automatically install dependencies like BepInEx)
3. Make sure all mods are enabled

## Step 3 — Configure your YAML

1. Open the Archipelago launcher
2. Go to **Generate Template Settings** and select Inscryption
3. This creates a `.yaml` file — open it in a text editor (Notepad is fine)
4. Set `name` to your slot name (e.g. `Adrian`)
5. Adjust any game options you want (difficulty, goal, etc.)
6. Send this `.yaml` file to the host

## Step 4 — Connect to the server

1. Launch Inscryption through r2modman (use **Start modded**)
2. On the main menu you will see an Archipelago connection screen
3. Enter the server address, your slot name (`Adrian`), and password if needed
4. Click **Connect**, then start a new game

## Tips

- Always launch through r2modman, not directly through Steam, or the mod won't load
- Finishing Act 1 is usually the main goal — check your YAML settings for details
- Items received may appear as cards or resources at the start of battles
