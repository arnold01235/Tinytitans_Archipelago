# Eskil — Resident Evil 3

## What you need

- **Resident Evil 3** (the 2020 remake) installed via Steam
- The **Archipelago client** installed: [github.com/ArchipelagoMW/Archipelago/releases](https://github.com/ArchipelagoMW/Archipelago/releases)

> **Note:** Double-check with the host which version of RE3 is being used — the 2020 remake or the original 1999 game. This guide covers the **2020 remake**.

## Step 1 — Install the RE3 Archipelago mod

1. Download the RE3 mod from the Archipelago releases page:
   [github.com/ArchipelagoMW/Archipelago/releases](https://github.com/ArchipelagoMW/Archipelago/releases)
2. Follow the included instructions — files typically go into the game's installation folder:
   - `Steam\steamapps\common\RESIDENT EVIL 3\`
3. A mod loader may be required — follow the mod's readme for details

## Step 2 — Configure your YAML

1. Open the Archipelago launcher
2. Go to **Generate Template Settings** and select Resident Evil 3
3. Open the `.yaml` file in a text editor
4. Set `name` to your slot name (e.g. `Eskil`)
5. Adjust difficulty and goal options as preferred
6. Send the `.yaml` file to the host

## Step 3 — Connect to the server

1. Launch the game with the mod installed
2. A connection dialog should appear (or check the in-game mod menu)
3. Enter the server address, your slot name (`Eskil`), and password if required
4. Connect and start a new game

## Tips

- Key items and weapon upgrades in the world are your checks
- Items received from the multiworld may appear in item boxes or be granted automatically
- If you disconnect, reconnect from the mod menu — your save progress is kept locally
